인피니티 플러스사각타입원형타입

인피니티 플러스

식약처 인증 의료기기, 압력 3ATA가 적용된 좌식 타입(Sitting type) 치료기
2인 치료의 새로운 기준 앉아서 사용하는 고압산소치료기

  • 수용 인원 : 2명
  • 식약처 인증 압력 : 3 ATA
  • 최대 테스트 압력 : 7 ATA
  • 크기 : 2,186㎜(L) × 853㎜(W) × 2,155㎜(H)
  • 재질 : 탄소강 + 아크릴
  • 기본 제공 옵션 : 리클라이너, 소화시스템, 비상시 수동운전 전환 가능, 정전 시 UPS 작동 시스템, 호흡기체 자동전환, 공기가압방식, 챔버 내부 독립호흡시스템(BIBS), 자동화 시스템, 자동음성 안내 시스템, 치료테이블 자동저장 시스템, 냉난방 시스템, 엔터테인먼트 시스템, EES(Easy Equalizing System)

제품 사진

※ 이 제품은 '의료기기'이며 '사용상의 주의사항'과 '사용방법'을 잘 읽고 사용하십시오.

사각타입

사각 Room Type으로 실내 공간 사용 최대

넓고 안락한 내부 공간과 식약처 승인 치료 압력 사용
내연성 재질을 사용한 리클라이너 의자, 시트, 베개 사용

  • 수용 인원 : 다인용 (Customizing)
  • 식약처 인증 압력 : 3 ATA
  • 크기 : 설치 여건에 따라 변경 가능
  • 재질 : 탄소강 SA516-70
  • 소화 시스템, CCTV, 믹스가스 시스템, 이중격실 설치 가능, 리클라이너 설치 가능, 비상시 수동운전 전환 가능, 정전 시 UPS 작동 시스템, 호흡기체 자동전환, 공기가압방식, 챔버 내부 독립호흡시스템(BIBS), 자동화 시스템, 자동음성 안내 시스템, 치료테이블 자동저장 시스템, 개별냉난방 시스템, 엔터테인먼트 시스템, EES(Easy Equalizing System)

  • 식약처 인증 압력 : 6 ATA
  • 최대 테스트 압력 : 16 ATA
